## 📄 О проекте
Адаптивный многостраничный сайт для студии дизайна, свёрстанный по макету из Figma. 
Включает главную страницу, страницу проекта и страницу «О студии».

## 🛠 Технологии
- HTML5
- CSS3 (Flexbox, Grid)
- Анимации
- Figma
- JavaScript (меню)

## ✨ Особенности
- Полная адаптация под мобильные устройства
- Семантическая разметка
- Все интерактивные элементы работают


## 📂 Как запустить
1. Склонируйте репозиторий:  
   `git clone https://github.com/vrust00/Modern-Brand-Design-Studion.git`
2. Откройте файл `index.html` в любом браузере.
